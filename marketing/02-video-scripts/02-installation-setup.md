## Step 1: Installation
**[TIME: 0:20-1:30]**

**[NARRATOR]**
"First, let's install the plugin. In your WordPress dashboard, go to Plugins, then Add New."

**[SHOW]**
Screen recording: Plugins → Add New

"In the search box, type 'WB Ad Manager'."

**[SHOW]**
Typing in search box, plugin appearing

"You'll see WB Ad Manager by WBcom Designs. Click Install Now."

**[SHOW]**
Click Install Now button

"Once it's installed, click Activate."

**[SHOW]**
Click Activate button

"Great! The plugin is now active. You'll see a new menu item called 'WB Ad Manager' in your sidebar."

**[SHOW]**
Highlight the new menu item

---

## Step 2: Setup Wizard
**[TIME: 1:30-2:30]**

**[NARRATOR]**
"When you first activate, you'll see the Setup Wizard. This walks you through the basic configuration."

**[SHOW]**
Setup wizard welcome screen

"Click 'Start Setup' to begin."

**Step 2a: Basic Settings**
"First, choose whether to show an ad label - that's the small text that says 'Advertisement'. Most sites use this for transparency."

**[SHOW]**
Ad label options

"Next, decide if you want to hide ads from logged-in administrators. I recommend turning this on so you can see your site without ads while working."

**[SHOW]**
Toggle the admin setting

**Step 2b: AdSense (Optional)**
"If you use Google AdSense, enter your Publisher ID here. You can find this in your AdSense dashboard - it starts with 'pub-'. If you don't use AdSense, just skip this step."

**[SHOW]**
AdSense field

"Click 'Next' to continue."

---

## Step 3: Create Your First Ad
**[TIME: 2:30-4:00]**

**[NARRATOR]**
"Now let's create your first ad. Click on WB Ad Manager in the sidebar, then click 'Add New'."

**[SHOW]**
Click Add New

**Step 3a: Title**
"Give your ad a name. This is just for your reference - visitors won't see it. I'll call this 'Sidebar Banner'."

**[SHOW]**
Type title

**Step 3b: Ad Type**
"Choose your ad type. For this example, I'll select Image Ad."

**[SHOW]**
Select Image Ad

"Now I'll upload my banner image. Click 'Upload Image' and select a file from your media library or upload a new one."

**[SHOW]**
Upload image process

"Add a destination URL - this is where visitors go when they click the ad."

**[SHOW]**
Enter URL

**Step 3c: Placement**
"Scroll down to Placement. This is where your ad will appear. I'll select 'Widget' since I want this in my sidebar."

**[SHOW]**
Select Widget placement

**Step 3d: Publish**
"Finally, click Publish."

**[SHOW]**
Click Publish button

"Congratulations! Your first ad is created."

---

## Step 4: Add Widget to Sidebar
**[TIME: 4:00-4:40]**

**[NARRATOR]**
"Since we chose Widget placement, we need to add the WB Ad Manager widget to a sidebar. Go to Appearance, then Widgets."

**[SHOW]**
Appearance → Widgets

"Find the WB Ad Manager Widget and drag it to your sidebar. Or click to add it if you're using the block widget editor."

**[SHOW]**
Add widget

"Select the ad we just created from the dropdown."

**[SHOW]**
Select ad

"Save the widget, and now let's check our site."

**[SHOW]**
View the frontend - ad is displayed!
