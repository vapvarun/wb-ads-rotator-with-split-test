## Video Recording Tips

### Technical Settings
- Resolution: 1920x1080 minimum (4K preferred)
- Frame rate: 30fps for tutorials, 60fps for smooth scrolling
- Format: MP4 (H.264)
- Browser: Chrome (clean profile, no extensions visible)
- WordPress: Fresh install with default theme or clean theme

### Best Practices
1. Clear browser cache before recording
2. Hide bookmarks bar
3. Use a clean WordPress installation
4. Create sample content ahead of time
5. Practice the flow before recording
6. Record in short segments for easy editing
7. Pause between actions for editing flexibility
8. Use zoom for small interface elements

### Content Guidelines
- Avoid personal data in screenshots
- Use placeholder/sample content
- Keep URLs and names professional
- Ensure ads shown are appropriate
- Remove any watermarks or distracting elements
