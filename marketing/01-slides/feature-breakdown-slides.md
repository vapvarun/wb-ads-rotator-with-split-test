# MODULE 1: DISPLAY ADS

## Slide 1.1: Ad Types You Can Create

**Headline:** Create Any Type of Ad

| Ad Type | What You Can Do |
|---------|-----------------|
| **Image Ad** | Upload banners, add click links, perfect for sponsors |

## Slide 1.2: Placement Options

**Headline:** 14+ Ways to Display Your Ads

**Automatic Placements (No Code Required):**

- Footer (bottom of every page)

- After paragraph X (configurable)
- Between archive posts
- In comment sections

**Advanced Placements:**
- Sticky bars (top/bottom)
- Floating corners
- Popup modals (time/scroll/exit triggers)
- Widget sidebars

**Community Placements:**
- BuddyPress activity feed

- bbPress forums & topics

## Slide 1.3: Smart Rotation

**Headline:** Multiple Ads, One Spot

**How It Works:**
1. Assign multiple ads to the same placement
2. Set priority for each ad (1-10)
3. Plugin automatically rotates them

**Use Cases:**
- A/B test different creatives
- Rotate sponsor ads fairly
- Show seasonal promotions
- Fill unsold inventory with AdSense
