## Template 4: Integration Announcement

**Subject options:**
- WB Ad Manager now works with [Plugin Name]
- New integration: [Plugin Name] + WB Ad Manager
- [Plugin Name] users: new ad placements available

---

**Body:**

Hey,

Great news if you use [PLUGIN NAME]:

## WB Ad Manager + [Plugin Name] Integration

We just released native integration with [Plugin Name]. Here's what it enables:

**New placement options:**
- [Placement 1]
- [Placement 2]
- [Placement 3]

**How it works:**
If [Plugin Name] is active on your site, these placements appear automatically in the Ad creation screen. No extra configuration needed.

**Get started:**

1. Make sure you're on the latest version of WB Ad Manager
2. Go to WB Ad Manager → Add New
3. Scroll to Placement options
4. You'll see the new [Plugin Name] placements

That's it.

---

Not using [Plugin Name]? No changes on your end - this update adds to the plugin without affecting existing features.

The WBcom Designs Team
