### Key Features

= Ad Management =

Create any type of ad:
* **Image ads** - Upload banners of any size
* **Rich content** - Full WYSIWYG editor
* **HTML/Code** - Custom scripts and embeds
* **AdSense** - Seamless Google integration
* **Email capture** - Newsletter signup forms

Display ads anywhere:
* Before or after post content
* After specific paragraphs
* In sidebar widgets
* As popups (exit intent, timed, scroll)
* Sticky header or footer bars
* Between archive posts
* In BuddyPress activity streams
* In bbPress forum topics

Target precisely:
* Device type (desktop, tablet, mobile)
* User status (logged in/out)
* User roles
* Specific post types
* Categories and tags
* Schedule by date, day, time

= Affiliate Link Management =

Turn ugly affiliate URLs into clean, trackable links:

* **Link cloaking** - yoursite.com/go/product-name
* **Click tracking** - Know which links perform
* **Broken link detection** - Catch dead links early
* **Bulk import** - Import existing links via CSV

= BuddyPress & bbPress Integration =

Native ad placements for community sites:

* Activity stream ads between posts
* Member profile page ads
* Forum topic ads between replies
* Full targeting support

## Installation

### Automatic Installation

1. Go to Plugins → Add New in your WordPress admin
2. Search for "WB Ad Manager"
3. Click "Install Now"
4. Click "Activate"

### Manual Installation

1. Download the plugin ZIP file
2. Go to Plugins → Add New → Upload Plugin
3. Select the ZIP and click "Install Now"
4. Click "Activate"

### After Activation

The setup wizard will guide you through initial configuration. Or skip it and start creating ads immediately - the plugin works with default settings.

## Frequently Asked Questions

### Does this work with my theme?

Yes. WB Ad Manager uses standard WordPress hooks and widgets. It's compatible with any properly-coded theme.

### Will this slow down my site?

No. The plugin is optimized for performance with lazy loading, efficient database queries, and minimal frontend JavaScript.

### Is there a limit on ads or links?

No limits. Create as many ads and cloaked links as you need.

### Does this work with BuddyPress?

Yes. When BuddyPress is active, additional placement options appear automatically. No extra configuration needed.

### Does this work with bbPress?

Yes. When bbPress is active, forum topic and reply placements become available automatically.

### Can I use this with page builders?

Yes. Use the WB Ad Manager widget or shortcodes with Elementor, Beaver Builder, Divi, and other page builders.

### Is the Pro version required?

No. The free version is fully functional for most sites. Pro adds advanced features like classifieds, advertiser portals, A/B testing, and detailed analytics.

### How do I get support?

Free users can post on the WordPress.org support forum. Pro users receive priority email support.
